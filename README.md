# How To Start

```
# install python3
sudo apt install python3

# clone repo
git clone https://github.com/htl-leonding-tadeot/face-pong.git
cd face-pong

# make env
python3 -m venv .venv

# for windows
.venv\Scripts\activate

# Linux/MacOS
source .venv/bin/activate

# install requirements
pip install -r requirements.txt

# run the project
python3 game.py
```
